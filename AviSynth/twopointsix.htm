AviSynth 2.6

New Video Features

  • New color formats: YV24 (24 bit), YV16 (16 bit), YV411 (12 bit) and Y8 (8 bit; greyscale). For a description of the color formats AviSynth can deal with see here. These formats are supported in most filters (including in source filters like AviSource and DirectShowSource).

New filters

  • A lot of filters and script-functions are added or improved. See the changelist for a complete list of these.

Plugins

  • One of the features of AviSynth 2.6 is that 2.5 plugins (plugins compiled with plugin api v3) can still be used in AviSynth 2.6, although you need to make sure you use it properly since it doesn't know about the new features in AviSynth 2.6. So when processing a clip (with say color format YV16) with a 2.5 plugin you will get a messed up clip at best, and a crash at worst, since YV16 is not supported in the v3 api.
