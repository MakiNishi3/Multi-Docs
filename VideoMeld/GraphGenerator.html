Graph Generator

Location: f(x) button on Audio Effect Editor window or Video Effect Editor

Use the Graph Generator to create a settings graph based on simple mathematical equations. The From and To times set the time range (in file time) to generate the graph. Use Interval to generate points on a fixed time interval. Use Points to generate a certain number of points (up to 10000).

Select one or two function to generate points. The functions may be added or multiplied together.

Use Initial t to set the initial time for the functions.
