Set Length

Menu Command: Edit | Set Length

Sets the length of all selected sections to a given time. The time can be entered or based on the longest or shortest section currently selected.

If the time is to be entered, enter the time into the box, then select how the time is to be applied.

Use Absolute length to set the length of all sections to the entered time. Use Add length to increase the length of the sections by the entered time. File based sections cannot be increased beyond their maximum length. Use Subtract length to decrease the length of the sections by the entered time. Be aware that subtracting time from a section could cause the section to become too short. A warning will appear if that happens.
