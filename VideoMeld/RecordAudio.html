Audio Recorder

Menu Command: Record | Audio Recorder

Audio Recorder captures audio on any input in the computer. Use the Input drop-down list to select the audio input for recording. The recording sampling rate and channels are shown below the input volume.

Use the Volume control to adjust the input level. The level should peak in the yellow or orange during recording. If the meters touch full scale, the volume is too high and clipping distortion may occur. Recording should be restarted with a lower volume.

Recordings are named using the Base filename, the current date and time (if Automatic file naming is checked), and the file type for the name. Click on the Base filename edit box to change the base filename and folder where the file is saved.

To automatically add the recording to the project, check the Add to project after recording box.

Use Synchronize with playback to start playing the project from the edit marker when recording is started. Use this feature to record different instrument tracks in the same song, such as recording a vocal track while playing music already added to the project.

Use Add Another Recorder to add another recorder block that can be assigned to a different input. This allows you to record from multiple inputs at the same time. Use the trash can icon in the upper right corner of a recorder block to remove a recorder.

Use Start All to start all recorders at the same time. Use Stop All to stop all recorders.
