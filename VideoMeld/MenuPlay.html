Play Menu Commands

Play commands start or stop playback of the project. Any muted tracks or sections are not played. If a track is selected as solo [1], only that track is played.

Play All
Plays the entire project from the beginning.

Resume
Continues playback from the playback marker.

From Marker
Starts playback at the edit marker's position and plays to the end of the project.

To Marker
Plays the part of the project shown in the view and stops at the edit marker's position.

View
Plays the part of the project shown in the view.

Stop
Stops playback.

Selection
Plays only the selected sections.
