Speech Section Settings

Menu Command: Track | Add Speech or Options | Section Settings when a speech section is selected or right-click a speech section and choose Settings or choose the Settings button in the section's title bar.

A speech section uses text-to-speech technology to convert text to spoken audio. Use speech sections to add temporary narrations to video or music or for estimating how long a document would take to read.

Use Speech Section Settings to change the following:

  • Text to be spoken
  • Text shown in the title bar
  • Voice rate
  • Voice pitch
  • Playback speed
  • Colour of the section's title bar
  • Channel configuration
Location
Shows the location on the hard drive where the audio file is stored. This setting cannot be changed.
Title
Sets the text shown in the title bar.
Voice rate
Sets the speed that the text will be spoken. Values below 0 decrease the speed. Values above 0 increase the speed.
Pitch
Sets the pitch of the voice. Values below 0 decrease the pitch. Values above 0 increase the pitch.
Speed
Sets the playback speed. Values above 100 increase the speed and pitch. Values below 100 lower the speed and pitch.
Channels
Sets the channel mapping for the section. The mapping is shown in the window below the setting.
