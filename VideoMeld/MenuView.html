View Menu Commands

View commands allow you to zoom in and out of the project. Use these commands to see a more detailed view of individual sections or a better view of the overall project. To quickly zoom in and out, use Ctrl+Up and Ctrl+Down arrow keys or the mouse wheel.

All
Displays the entire project.

Zoom In and Zoom Out
Zoom In expands the time line showing more detail. Zoom Out compresses the time line showing less detail, but more of the overall project. Zooming is centered on the edit marker's current position. Move the marker to the area of interest before zooming.

1 Second
Displays one second of the project. At this level, you can easily see small sections and perform accurate editing.

10 Seconds
Displays ten seconds of the project. This level is useful for manipulating small sections.

1 Minute
Displays one minute of the project. This level give you a good overview of short projects.

5 Minutes
Displays five minute of the project. This level is good for music and songs.

1 Hour
Displays 1 hour of the project, which is useful for CD audio projects.

Track Height
Displays a submenu for setting the track height.

Section Detail
Sets the level of detail shown in sections.

  • None: Displays no file content in sections. On slow systems this setting significantly reduces processing, speeding up the overall user interface.
  • Low: Displays single frame only (thumbnail). Reduces the amount of processing required to display video and image sections by displaying only the first frame within the section.
  • High: Displays multiple frames (storyboard). Requires significant processing to open, seek, and display video frames throughout video sections. Not recommended when using the Follow Playback option below.

Beginning Of Section and End Of Section
Moves the view to the beginning or end of the selected sections. Use these commands to quickly see the previous or next section when zoomed in.

End Of Track
Moves the view to the end of the selected track.

Marker
Moves the view to the edit marker if you are zoomed in and the marker is outside the view.

Follow Playback
When playing the project and zoomed in, the time line scrolls to keep the playback marker in the center of the window. Following playback requires significant processing for large projects. Set Section Detail above to Low or less to reduce the processing required.

Grid
Sets the grid spacing and related settings.

Snap To Grid
Displays a grid and snaps the marker to the grid. When dragging a section, the starting point of the section is snapped to a grid line. The grid is visible only if the space between each line can be shown, otherwise you will need to zoom in to see the grid.
