Options Menu Commands

Section Settings
Shows settings for the selected section. This command may be used only when one section is selected. All sections have a button in the upper left corner to display settings. Settings for sections include:

  • Audio Settings
  • Caption Settings
  • Image Settings
  • Overlay Settings
  • Video Settings

Track Settings
Shows settings for the selected track.

Playback Settings
Shows playback settings, such as preview resolution, frame rate, and audio playback device.

Project Settings
Shows project settings, such as aspect ratio and number of channels.

Configure VideoMeld
Changes setting in VideoMeld, such as the mouse wheel function, mouse rendering, window colour scheme, and video cache size.

Check For Updates
Periodically checks the website for new versions of VideoMeld.
