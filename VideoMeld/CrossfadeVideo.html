Crossfade Video

Use Effect | Fade Graph to display fade points in the section, then use the mouse to add or adjust points or select a preset from Effect | Preset Graphs. Or use Crossfade Sections to automatically set the fade points based on the overlap. The video section must be in the top layer (last track).

Manual crossfade step-by-step:

  1. Add Video 1 to Track 1.
  2. Create a new track.
  3. Add Video 2 to Track 2.
  4. Drag Video 2 to a position so that it starts about 2 seconds before the end of Video 1.
  5. Use Effect | Fade Graph to display fade graphs, if not already shown.
  6. Use Effect | Preset Graphs | Fade In 2.0s

Automatic crossfade step-by-step:

  1. Add Video 1 to Track 1.
  2. Create a new track.
  3. Add Video 2 to Track 2.
  4. Drag Video 2 to a position so that it overlaps the end of Video 1 by a few seconds.
  5. Select both Video 1 and Video 2.
  6. Use Effect | Crossfade Sections.
