Presets

Presets store settings in a file so they can be recalled again the next time the effect is used. Controls for presets consist of a drop down list box, a [+] button, and a [-] button.

To add a new preset:

  1. Enter in all the new settings.
  2. Choose the [+] button.
  3. Type in a new name for the preset in the drop down list.
  4. Choose OK.

To delete a preset:

  1. Select the preset from the drop down list.
  2. Choose the [-] button.

To change a preset:

  1. Select the preset from the drop down list.
  2. Change the settings, as required.
  3. Add the preset, as above. It will overwrite the old preset.
