Keyboard Commands

In addition to menu keystrokes, such as Alt+F O to open a project, VideMeld includes the following keystrokes.

Keystroke Action
` (open quote) Displays the Set Marker window.
A Plays the entire project.
Ctrl+A Selects all sections.
Shift+A Zooms out to display the entire project.
Ctrl+C Copies selected sections into the clipboard.
Ctrl+G Shows or hides the grid and enables or disables the snap-to-grid feature.
Ctrl+M Mutes the selected sections.
Ctrl+O Opens a sound.
Ctrl+S Saves the project.
Ctrl+V Pastes the sections in the clipboard into the project at the marker's position.
Ctrl+X Cuts the selected sections from the project and copies them into the clipboard.
Ctrl+Z Undoes the last modification.
Ins Adds a new file section.
Del Deletes the selected sections.
Shift+Ins Creates a new track, inserting it after the selected one.
Shift+Del Deletes the selected track.
Tab Moves the view to the marker's position if it is off screen.
Space Starts or stops playback at the marker's position.
Ctrl+Up Zooms in.
Ctrl+Down Zooms out.
Alt+Up Moves the selected track up.
Alt+Down Moves the selected track down.
Left Scrolls the view left.
Right Scrolls the view right.
Page Up Scrolls the view left one page/screen.
Page Down Scrolls the view right one page/screen.
Home Moves view to the beginning of the selected section.
End Moves view to the end of the selected section.
Ctrl+End Moves view to the end of the selected track.
F1 Displays help.
1,2,4,5,9,0 Set zoom level to 1 second, 10 seconds, 1 minute, 5 minutes, or 1 hour respectively.
