Crossfade Audio

Use Effect | Fade Graph to display fade points in the section, then use the mouse to add or adjust points or select a preset from Effect | Preset Graphs. Or use Crossfade Sections to automatically set the fade points based on the overlap.

Manual crossfade step-by-step:

  1. Add Sound 1 to Track 1.
  2. Create a new track.
  3. Add Sound 2 to Track 2.
  4. Drag Sound 2 to a position so that it starts about 2 seconds before the end of Sound 1.
  5. Use Effect | Fade Graph to display fade graphs, if not already shown.
  6. Use Effect | Preset Graphs | Fade In 2.0s

Automatic crossfade step-by-step:

  1. Add Sound 1 to Track 1.
  2. Create a new track.
  3. Add Sound 2 to Track 2.
  4. Drag Sound 2 to a position so that it overlaps the end of Sound 1 by a few seconds.
  5. Select both Sound 1 and Sound2.
  6. Use Effect | Crossfade Sections.
