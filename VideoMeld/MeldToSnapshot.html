Meld Project To Snapshot Settings

Menu Command: File | Meld Project To Snapshot

Combines all video sections and a given instant in time into a single image file. After entering a filename and selecting the image type to use, the following image settings are presented.

Resolution
Sets the horizontal and vertical size in pixels of each frame in the video. For best results, the resolution's aspect ratio should match the project's set under Options | Project.
Snapshot time
Sets the time within the project to create the image. Use the track bar below the video preview window to find the time required or enter the time directly.
