Auto Split Video

Menu Command: Edit | Auto Split Video

Divides a long video section into smaller sections based on content changes (shot or scene changes) using a relatively simple detection algorithm. Use the Mode setting to control how splits are generated. Select Automatic to generate all splits automatically (not recommended). Select Interactive mode to see where content changes are detected, then use the Split button to set a split or the Next button to continue to the next detected content change.

Use Maximum rate of change to control the amount of change from one frame to the next to be considered a shot or scene change. Values around 25% usually work well. Use lower values when all content is very similar with smooth camera movements. Use higher values for rapid camera movements or quickly varying colours or contrast.

Due to the simplicity of the detection algorithm, some content changes may be missed (gradual crossfades, for example) or false positives may occur (shakey camera, for example). Interactive mode is recommended with a slightly lower Maximum rate of change value to detect and verify the appropriate split points.

The progress bar shows the amount of the video file process so far and the relative positions of the splits. The total number of splits is given to the right.

Choose the Apply Splits button to apply all the splits to the video section.
