New Project Settings

Menu Command: File | New Project

Use New Project to start a new project and set the aspect ratio of the video.

Common aspect ratios
Allows you to quickly select a commonly used aspect ratio.
Video aspect ratio
Sets the aspect ratio for the project. The aspect ratio is the ratio of the width to the height of the video. A high definition video with a width and height of 1920 by 1080 has an aspect ratio of 16 to 9 (1920 / 1080 = 1.777... = 16 / 9). More information is available on Wikipedia.
