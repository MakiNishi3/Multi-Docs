Audio Section Settings

Menu Command: Options | Section Settings or right-click an audio section and choose Settings or choose the Settings button in the section's title bar.

Use Audio Section Settings to change the following:

  • Text shown in the title bar
  • Time range of audio played from the file
  • Playback speed
  • Colour of the section's title bar
  • Channel configuration
  • Decode a variable bitrate file for better synchronization
Location
Shows the location on the hard drive where the audio file is stored. This setting cannot be changed.
Decode Audio To Wave File For Best Synchronization
Converts an encoded, variable bitrate audio file to PCM Wave format, which supports sample accurate seeking and synchronization. Variable bitrate files are often difficult to synchronize when previewing. Decoding to Wave eliminates synchronization problems. Note that some video files have poor audio/video synchronization due to differences in hardware timing. A very slight adjustment to the Speed setting (below) will correct a synchronization drift.
Title
Sets the text shown in the title bar.
Range
Sets the start and finish times within the file. Choose the All button to play the entire file.
Speed
Sets the playback speed. Values above 100 increase the speed and pitch. Values below 100 lower the speed and pitch. To compensate for pitch changes, add a Pitch effect.
Channels
Sets the channel mapping for the section. The mapping is shown in the window below the setting. Either select a preset channel mapping from the drop-down list or manually connect source channels to destination channels. To connect channels manually, first click on a source channel circle, then click on each of the destination channel circles. If the connection already exists, clicking on the destination channel circle will remove the connection.
Connect available channels only
Only channels that are available in the project setting are connected. Otherwise all channels are connected in case different project channel settings are used later.
