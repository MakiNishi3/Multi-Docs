Video Effects List

Menu Command: Effect | Add Or Modify then, Add Effect button (with one video, image, caption, or overlay section selected)

Lists all available video effects categories in a collapsible list structure. Expand a list category to see effects available in that category.

Use Placement to set the starting point for the effect.

Use Duration to set the duration of the effect.

Usually transitions and fades are added to the end of the video by default.

Choose an effect, then choose the Add button to add it to the list of active effects.
