File Menu Commands

New
Creates a new project with a single blank track. If another project is opened, you are asked to save any changes before the new project is created.

Open
Opens an existing project.

Save
Saves the project to the disk. The names and positions of all of the files used in the project are saved. Note that any changes made to those files may change the project the next time it is opened.

Save as
Saves the project under a different name.

Meld Project To Audio File
Combines and mixes all audio sections of the project into a single audio file.

Meld Project To Video File
Combines and mixes all video and audio sections of the project into a single video file.

Meld Project To Snapshot
Combines and mixes all video sections at a given instant of time into a single image file.

Collect All Files In Project
Copies all files used in the project into a single folder.

Exit
Exits VideoMeld.
