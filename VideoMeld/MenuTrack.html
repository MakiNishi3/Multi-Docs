Track Menu Commands

Track menu commands create new tracks, add files (sections) to the selected track, change the order of tracks, or delete the selected track.

Add Track
Creates a new, blank track after the currently selected one.

Move up and Move down
Moves the selected track up or down. This lets you arrange the order of tracks, which changes the way videos are layered.

The following Add commands add files or sections to the selected track. Sections are added at the marker, if there is room, or appended to the end of the track if there isn't enough room.

Add File
Adds a file to the selected track. The file can be audio, video, or an image. All supported file types are listed.

Add Audio
Adds an audio file. Only supported audio files are listed.

Add Video
Adds a video file. Only supported video files are listed.

Add Image
Adds an image file or photo. Only supported image files are listed. Add a Fade video effect to create a transparent logo or watermark.

Add Caption
Adds a text caption or scrolling text.

Add Overlay
Adds an overlay. Overlays are useful for applying additional effects over the melded video or limiting effects to a smaller region of the video.

Add Speech
Adds a text-to-speech section for converting text to spoken audio.

Delete Track
Deletes the selected track and all sections in that track.
