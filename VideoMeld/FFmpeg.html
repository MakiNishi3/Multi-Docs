Using FFmpeg Modules

FFmpeg is a separate multimedia framework. It includes avcodec and avformat modules, which handle many audio and video files formats such as MPEG-2, AC3, VOB, and more. Configuring VideoMeld to use FFmpeg modules greatly increases the number of file formats VideoMeld can open.

Due to patent and licensing issues, FFmpeg modules cannot be included with VideoMeld or even hosted on the website. Please do not contact GoldWave Inc. to request copies of the modules. The modules have to be downloaded and installed separately. See legal requirements below. Please avoid using patented audio and video formats when possible. Use Opus or FLAC for audio.

VideoMeld Requirements

VideoMeld requires version 2 of the FFmpeg or Libav modules (Windows 64 bit DLLs). Specifically VideoMeld requires the following files: 

	avcodec-56.dll
	avformat-56.dll
	avutil-54.dll
	swresample-2.dll  or  avresample-1.dll
	swscale-3.dll
To use FFmpeg (or Libav) in VideoMeld, simply use Options | Configure to set the folder where the files are located on the computer or use the Download button to automatically download and install the Libav LGPL version of the modules.

Legal Requirements

GPL:
Since VideoMeld is a closed source program, only LGPL versions of the modules may be used. Using GPL versions, such as the Shared Zeranoe FFmpeg Build or Libav Win32 Build is not permitted under GPL. At the Zaranoe site you can request a LGPL version (for a fee) or you can search the Internet for other alternatives, provided they have the above files. Always follow safe practices when downloading files from the Internet.

Patents:
VideoMeld does not include any patent licenses not already covered by the operating system (see formats covered in Windows 7 or later, which includes AAC, H.264, and others). Depending on the kind audio or video being decoded, additional patent licenses may be required. Anyone installing the above modules to use formats not directly handled by VideoMeld are responsible for ensuring they meet all third-party patent license requirements of those formats.

Given the number of patents and the complex jurisdictional issues, GoldWave Inc. cannot answer questions related to patents.
