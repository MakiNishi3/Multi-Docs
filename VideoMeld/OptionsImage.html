Image Settings

Menu Command: Options | Section Settings or right-click an image section and choose Settings or choose the Settings button in the section's title bar.

Use Image Settings to set the display time and title bar text and colour.

Location
Shows the location on the hard drive of the image file. This setting cannot be changed.
Resolution
Shows the resolution of the image. This setting cannot be changed.
Title
Sets the section's title bar text.
Display time
Set the length of time the image is shown in the video. The default time is 5 seconds. To change the default, use Project Settings.

Use Video Effects to add special effects to the image or to change it's size and position.
