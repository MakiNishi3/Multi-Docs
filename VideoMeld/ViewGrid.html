Grid Settings

Menu Command: View | Grid

Use Grid Settings to display a grid within tracks or set the grid spacing and divisions. The amount of time between each major grid line is set using either the Beats per minute (BPM) value or the Spacing time value. A BPM value of 120 draws major grid lines at 0.5 second intervals. A Spacing value of 5 draws major grid lines at 5 second intervals.

Divisions sets the number of divisions between each major grid line. Minor grid lines (dashed) are drawn between major grid lines for each division. If Divisions is 1, no minor grid lines are drawn. A value of 4 divides the space between major grid lines into 4 parts and three minor grid lines are drawn.

Offset sets the starting time of the grid.

Check Show grid to make the grid visible in tracks. Check Snap to grid to align the marker and sections to major or minor grid lines when moved.
