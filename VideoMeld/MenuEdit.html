Edit Menu Commands

Before using some of these edit commands, make sure the correct section is selected. Some commands require that only one section is selected. See Getting Started.

Undo
Reverses the last modification.

Redo
Reverses the last undo.

Copy
Makes a copy of the selected sections in the clipboard.

Cut
Copies the selected sections to the clipboard and removes them from the project.

Paste
Pastes sections from the clipboard into the selected track at the marker's position. If there is no room for the sections, they are pasted at the end of the track. All track and section spacing is preserved. Sections must be copied before you can paste.

Delete
Removes all selected sections from the project and shifts all following sections to the left to fill the space. Use this to remove both the sections and time used by the sections, such as removing commercials.

Erase
Removes all selected sections from the project. Following sections are not shifted. Use this to remove sections without affecting the timeline after the sections, such as removing profanity from audio without affecting synchronization with the remaining video.

Compact Sections
Removes all spaces between selected sections so that sections are adjacent to each other. Use this after trimming to remove the gaps between sections.

Mute
Disables the selected sections so that they will not be melded or played. Use this command to see what the project is like without the selected sections.

Move
Moves the selected sections to an exact time.

Split
Divides the selected sections at the marker's position. Use this command to insert a section in the middle of the selected section or to move part of a section to a different time or track.

Trim Left
Removes the beginning of the selected sections up to the marker's position. Use this command to keep the last 20 seconds of a 30 second file, for example.

Trim Right
Removes the end of the selected sections at the marker's position. The part of the section extending from the marker's position to the end of the section is trimmed. Use this command to keep the first 20 seconds of a 30 second file, for example.

Auto Split Video
Automatically splits a video section into smaller sections based on content (shot or scene) changes.

Open In Editor
Opens the audio section's file in an audio editor. Use Configure VideoMeld to set the editor program.

Select All sections
Selects all sections in all tracks.

Select All Sections In Track
Selects all sections in the currently selected track.

Link Sections
Links together all currently selected sections so that if one section is selected, all are selected automatically. When a video is opened, the video and audio sections are linked by default. Use the Link icon in a section's title bar to unlink them or use Unlink Sections below. A section may be linked to only one group at a time. If a section is linked to a new group, it is unlinked from the previous group.

Unlink Sections
Disbands linked sections so they may be selected individually. The Link icon in a section's title bar allows individual sections to be removed from the group.

Move marker
Moves the marker to an exact time.

Space Equally
Equally spaces selected sections in the current track.

Find
Displays the Find bar to search for a selection of a given name.
