Project Settings

Menu Command: Options | Project Settings

Use Project Settings to set the project's aspect ratio, audio channels, and default image display time.

Video aspect ratio
Sets the aspect ratio for the video. The aspect ratio is the ratio of the width to the height of the video. A high definition video with a width and height of 1920 by 1080 has an aspect ratio of 16 to 9 (1920 / 1080 = 1.777... = 16 / 9). More information is available on Wikipedia. All video and images added to a project should have the same aspect ratio, otherwise distortion may occur. Add an Aspect Ratio Correction video effect to adjust the aspect ratio of a file that does not match the project's aspect ratio.
Audio channels
Sets the number of channels to use for the project. VideoMeld supports mono, stereo, 3.1 channel, 5.1 channel, and 7.1 channel arrangements. You must set this before adding audio sections to the project to ensure that channels are mapped appropriately.
Image display time
Sets the default display time for images and overlays. When adding a large number of images to create frames of animation, set this value to the amount of time for each frame. For example, to add images with a frame rate of 20 frames-per-second, set this to 0.05 (1/20).
Aspect ratio correction
Use this setting to automatically correct the aspect ratio for images and videos that differ from the project's aspect ratio. If the project is set to a 16:9 aspect ratio, but the video or image is 4:3, then this feature will scale the content to remove any distortion, so things don't look too wide or too thin. Select "Automatic" to allow VideoMeld to add the Aspect Ratio Correction effect automatically when a video or image is added. Select "Manual" to not add the effect.
