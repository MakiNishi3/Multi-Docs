Collect All Files In Project

Menu Command: File | Collect All Files In Project

Copies all files used in the project to a single folder. All video, audio, images, and related files are copied to the folder. An updated version of the project itself is saved in the same folder so that it uses the copied files rather than the original files.

Use this feature to create a portable version of the project for editing on another computer or to speed up previewing by copying networked files to a local folder.

Note that if some original files in different folders have the same name, those files will be given unique names in the single folder. Any files to be renamed are listed.
